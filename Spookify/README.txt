# How to run website:

1. Clone the repository

2. Install requirements
``` pip3 install -r requirements.txt ```

3. Start a postgres server

4. Run create_db.py to create the database

5. Run import_songs.py to import the songs into the database

6. Run main.py to start the website

7. Open a browser and go to http://127.0.0.1:5000/
